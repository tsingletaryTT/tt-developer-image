# Forge venv requirements for tt-developer-image.
#
# tt-forge and pjrt-plugin-tt declare URL deps in their package metadata.
# Both pip (resolution-too-deep) and uv (URL-in-transitive-metadata policy)
# refuse to resolve these unless they appear here as direct requirements.
#
# tt-forge-onnx is intentionally EXCLUDED: it requires torch==2.7.0 which
# directly conflicts with pjrt-plugin-tt's torch==2.10.0. They cannot coexist
# in the same venv. The primary use case of this image is the TT-XLA/JAX
# runtime (tt-forge + pjrt-plugin-tt), not the ONNX frontend.
# The forge/ Python module comes from tt-forge-onnx; without it, use pjrt_plugin_tt
# and jax directly.
#
# How to update when tt-forge is bumped:
#   1. Unzip the new tt-forge wheel and read dist-info/METADATA for URL deps
#   2. Unzip pjrt-plugin-tt and read its METADATA for URL deps
#   3. Add all URL deps below; resolve any torch version conflicts first
